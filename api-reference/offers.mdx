## List Offers

Returns paginated exchange rate offers available to clients.

```bash
GET /api/v1/partner/offers
Authorization: Bearer sk_personal_YOUR_KEY
```

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `page` | integer | Page number (default: 1) |
| `limit` | integer | Items per page (default: 25, max: 100) |
| `filters` | string | JSON-encoded TypeORM filters |
| `sort` | string | JSON-encoded sort array |
