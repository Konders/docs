## List Exchange Offices

Returns all publicly visible exchange offices with their current rates, schedule, and contact information.

```bash
GET /api/v1/partner/exchanges
Authorization: Bearer sk_personal_YOUR_KEY
```

### Response

```json
[
  {
    "id": 1,
    "name": "Main Office",
    "address": "Kyiv, Khreshchatyk 1",
    "schedule": [...],
    "offers": [...]
  }
]
```

---

## Get Exchange Office by ID

```bash
GET /api/v1/partner/exchanges/:id
Authorization: Bearer sk_personal_YOUR_KEY
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | integer | Exchange office ID |

### Response

Returns a single exchange office object. Returns `404` if not found.
